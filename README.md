# Rails Girls Signup

This is a site for potential candidates to Rails Girls.

* Ruby version: '2.3.0'

* Rails version: '5.0.0'  
gem install '5.0.0'

* Database: Postgres

* Deployment instructions  
$ `git clone git@github.com:jendiamond/railsgirls_signup.git`  
$ `bundle`  
$ `rake db:create`  
$ `rake db:migrate`
$ `rake db:seed`

* To run the test suite: `bundle exec rspec`
